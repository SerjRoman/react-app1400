1. npx create-react-app название --template typescript
2. всё что в папках src и public - УДАЛЯЕМ 
3. создаем index.html в папке public 
4. один <div> елемент с id = root. В этот елемент будет встраиваться весь реакт
5. в папке src создаем файл index.tsx (ВАЖНЫЙ ФАЙЛ  будет встраивать весь реакт в хтмл) 
6. создать переменную (константу) rootElement и туда получаем по айди див из хтмл as HTMLElement (type assertion)
7. еще одна переменная (не забыть импорт  createRoot from 'react-dom/client')
   в этой новой переменной используем импортированную функцию и передаем туда rootElement
8. новый файл (компонент) в папке src - App.tsx
9. создаем компонент App (функция с её експортом)
10. для отображения компонента в index.tsx делаем его рендер root.render(<App/>)
11. npm run start
12. Папка pages в src отвечает за компоненты, которые отображают всю страницу.
13. Папка hooks в src отвечает за файлы с кастомными хуками.
14. Папка shared в src отвечает за компоненты, которые являются общими для страниц.
